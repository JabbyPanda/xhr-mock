`xhr-mock` is a utility for mocking HTTP responses .

## [📖 Documentation](./packages/xhr-mock)

The user focused documentation can be found [here](./packages/xhr-mock).

## 🛠 Development

Install the dependencies:

```bash
yarn install
yarn run clean
yarn run lint
yarn run build
yarn run test
```
